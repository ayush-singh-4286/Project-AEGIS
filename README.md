## Overview
Project AEGIS is a rapidly deployable smart emergency shelter designed for disaster situations such as floods, cyclones, and heatwaves.

The shelter becomes operational within 15 minutes and provides:
- Emergency shelter
- Solar-powered electricity
- Lighting system
- Safety monitoring
- Communication support
- Clean water support

---

## Problem Statement
During natural disasters, affected people often lose access to:
- Safe shelter
- Electricity
- Communication systems
- Clean drinking water
- Emergency monitoring systems

Project AEGIS aims to solve these issues using a self-sustaining intelligent disaster relief system.

## Deployment Procedure
1. Unfold the shelter structure
2. Activate the power system
3. Start ESP32 monitoring
4. Enable sensors and alert system
5. Inflate flood-support base if required

---

## Project Impact
Project AEGIS helps disaster-affected communities by providing immediate shelter, safety monitoring, communication support, and emergency utilities during the critical early hours of disasters.
